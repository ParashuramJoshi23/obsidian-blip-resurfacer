# Claude Configuration Instructions

This directory contains configuration files to help Claude understand the project structure and aid in development.

## Project Structure
- **src/**: The source code of the project.
- **tests/**: All the test files for the project.
- **docs/**: Documentation files for users and developers.

## How to Use Claude
Claude can assist with various development tasks such as code reviews, generating documentation, and more. Use the configuration files in this directory to understand the context better.
